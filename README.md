# Honors Thesis

## Project Description
This project investigates the feasibility of continuous phishing detection using resource-efficient AI models integrated with computer vision–based input pipelines. The goal is to evaluate whether lightweight transformer models can accurately identify phishing content in near–real-time settings, such as monitoring email interfaces or visual inbox representations, while maintaining low computational overhead.

## Objectives
- Develop a baseline phishing detection pipeline using traditional NLP methods.
- Implement a lightweight transformer-based model for phishing classification.
- Simulate continuous monitoring via looping inference over streamed inputs.
- Establish a foundation for integrating computer vision (OCR) into phishing detection workflows.
